# 15 — Полезные контакты

**Компонент:** `pages/ResidentUsefulContactsPage.tsx` (`/useful-contacts`, алиас `/contacts`).
Фича за флагом `useful-contacts`.

## Что делает (функционал)

Справочник полезных контактов для жителя: УК, диспетчер, аварийные службы, охрана, партнёры. Фичи:

- Список контактов с телефоном (кнопка звонка `tel:`), категорией, описанием.
- Возможны экстренные/аварийные контакты отдельной группой.

## Креативное направление

Тёплый справочник. Группировка по категориям (Аварийные / УК / Сервисы) с заголовками-секциями.
Контакт — карточка-строка: иконка категории в кружке `--brand-tint`, название, должность/описание
`--text-secondary`, справа крупная круглая кнопка вызова (оранжевая, иконка трубки). Аварийные —
critical-акцент (красная иконка, плашка «24/7»).

**Анимации:** нажатие кнопки вызова — pulse; въезд карточек по секциям.
