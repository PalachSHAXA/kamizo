# 16 — Оценка сотрудников

**Компонент:** `pages/ResidentRateEmployeesPage.tsx` (`/rate-employees`, только resident).

## Что делает (функционал)

Житель оценивает сотрудников УК (исполнителей/персонал) — звёзды + отзыв. Фичи:

- Список сотрудников, доступных для оценки.
- Оценка звёздами (rating) + текстовый отзыв.
- Возможна благодарность конкретному сотруднику.

## Креативное направление

Тёплый «отзыв о сервисе». Список сотрудников — карточки: аватар (инициалы/фото), имя, специализация
(чип), текущий средний рейтинг звёздами. Тап → модалка/шит оценки: крупные **оранжевые звёзды** с
пружинным заполнением, поле отзыва, кнопка «Отправить». После отправки — благодарственная галочка.

**Анимации:** заполнение звёзд пружиной; отправка — галочка/сердечко; въезд карточек.

## Состояния

- Уже оценён — бейдж «оценено» + возможность изменить.
- Нет сотрудников для оценки — пустое состояние.
- Отправка — спиннер, блок повторной отправки.
