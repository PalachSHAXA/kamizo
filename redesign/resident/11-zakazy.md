# 11 — Мои заказы (маркетплейс)

**Компоненты:** `pages/MarketplaceOrdersPage.tsx` +
`pages/executor/components/MarketplaceOrderDetailsModal.tsx` (детали заказа).

## Что делает (функционал)

История и статусы заказов жителя из маркетплейса. Фичи:

- Список заказов со статусом (новый / в работе / в пути / выполнен / отменён).
- Позиции заказа, сумма, дата, исполнитель/курьер.
- Детали заказа с таймлайном статусов (MarketplaceOrderDetailsModal).

## Креативное направление

Лента заказов как «трек-посылок». Карточка заказа: номер, дата, превью позиций (мини-фото-стек),
сумма (`tabular-nums`), крупный бейдж статуса. **Таймлайн доставки** горизонтальный/вертикальный с
точками-стадиями (Оформлен → Принят → В пути → Доставлен): пройдено success, текущее оранжевый пульс.
Детали — модалка с полным таймлайном, позициями и кнопкой повтора/поддержки.

**Анимации:** прогресс таймлайна заполняется; обновление статуса — мягкая подсветка карточки;
въезд карточек.

## Состояния

- Нет заказов — пустое состояние + ссылка на витрину.
- Отменён — серый/desaturated + бейдж expired.
- Загрузка — скелетоны.
