# UK CRM - Обзор проекта

## Краткое описание

CRM-система для управляющих компаний (УК) Узбекистана - первый специализированный продукт на рынке, адаптированный под местную специфику.

## Ключевые особенности рынка

### Возможности
- Рынок практически свободен - нет локальных специализированных решений
- Уникальная возможность стать первым продуктом в нише
- Возможность захватить рынок до появления конкурентов

### Требования локализации
- Интеграция с Payme/Click (95% рынка платежей)
- Telegram-first коммуникации (18 млн пользователей в УЗ)
- Соответствие закону о локализации персональных данных (ЗРУ-547)

## Целевая аудитория

### Первичные клиенты
- Крупные застройщики (Murad Buildings, Grand Capital)
- Управляющие компании многоквартирных домов
- Девелоперы жилых комплексов

### Конечные пользователи
1. **Админы/менеджеры УК** - управление заявками, отчетность, аналитика
2. **Рабочий персонал** - выполнение заявок, фото-отчеты
3. **Жители** - подача заявок, оплата, коммуникация

## Болевые точки для решения

| Проблема | Влияние | Решение в CRM |
|----------|---------|---------------|
| Потеря заявок | 60% жителей недовольны | Единая система трекинга |
| Отсутствие контроля персонала | Высокая текучесть 30%+ | KPI и мониторинг |
| Низкая собираемость платежей | Рост задолженностей | Интеграция платежей + напоминания |
| Разрозненные каналы связи | Жалобы теряются | Telegram + Push + единый интерфейс |

| Тариф | Цена (за кв./мес) | Функции |
|-------|-------------------|---------|
| Start | $0.50-1.00 | Заявки, коммуникации, платежи |
| Pro | $1.50-2.00 | + Аналитика, автоматизация, API |
| Enterprise | Индивидуально | + Брендирование, SLA, интеграции |

### Дополнительная монетизация
- Комиссия с маркетплейса услуг (клининг, ремонт)
- Транзакционная комиссия за платежи (0.5-1%)
