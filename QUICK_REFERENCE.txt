SIDEBAR BADGES (NOTIFICATION COUNTS):
================================================================================

  /announcements  = Unread announcements
  /chat           = Unread chat messages

  /               = Active requests
  /announcements  = Unread announcements
  /meetings       = Upcoming meetings (7-day window)
  /chat           = Unread chat messages

  /requests       = New unassigned requests
  /meetings       = Upcoming meetings (7-day window)
  /chat           = Unread chat messages

Meeting Badge Colors:
  🔵 BLUE         = Voting open
  🟢 GREEN        = Confirmed/completed
  ⚪ NORMAL/PULSE = Upcoming

================================================================================
MULTI-TENANT ROUTING:
================================================================================

Super Admin Impersonation:
  - Can enter any tenant via /admin_url?auto_auth=...
  - Amber banner shown at top with tenant name
  - Exit button returns to super admin dashboard
  - Impersonation data stored in localStorage['kamizo_impersonation']

Feature Control:
  - Admin can enable/disable features per tenant
  - Disabled features shown with 🔒 lock icon in sidebar
  - Direct URL access to disabled features shows home page
